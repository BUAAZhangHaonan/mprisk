# Configs

This directory stores the canonical active configuration for the paper pipeline. Active config filenames describe their role and do not carry iteration dates or _vN suffixes. Schema IDs, immutable run IDs, model product names, and frozen dataset or delivery identifiers retain their source-defined versions.

Local absolute paths belong in `configs/local_paths.yaml`, copied from `local_paths.example.yaml`. That local file is ignored by git.
