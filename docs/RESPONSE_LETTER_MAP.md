# Response Letter Map

## Main Revision Story

The revision reframes the work as measurement and interpretation of multimodal pre-generation
state structure, followed by an independent analysis of its association with affective Misread.

The response letter should emphasize:

1. mental health is motivation, not a clinical claim;
2. `t0` is the operational audit window;
3. trajectory representation replaces the single-layer point view;
4. `S/D/R` describe complementary state dimensions rather than a single risk score;
5. State Patterns are mechanism-oriented configurations rather than error categories;
6. Conflict/Aligned representation learning is separate from the frozen-representation Misread probe;
7. response evaluation measures mitigation of affective misattribution, not correction of the
   Diagnostic Misread label.
